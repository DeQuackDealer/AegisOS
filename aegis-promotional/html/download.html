Installation Guide

Download ISO

Download your chosen edition ISO file from above. Verify the checksum if provided.

Create Bootable Media

Use tools like Rufus (Windows), balenaEtcher (cross-platform), or dd command (Linux/macOS) to create a bootable USB drive.

Boot from USB

Restart your computer and boot from the USB drive. You may need to change boot order in BIOS/UEFI settings.

Install Aegis OS

Follow the installation wizard to install Aegis OS on your system. Choose your preferred partitioning and user settings.

Activate License

For paid editions, enter your license key during or after installation to unlock premium features.
