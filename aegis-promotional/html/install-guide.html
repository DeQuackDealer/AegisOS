Installation Guide

Step-by-step guide to install Aegis OS on your system

üöÄ Installation Steps

1
Download & Verify ISO
Download Aegis OS ISO (2.1 GB) from the official website and verify the checksum using SHA-256.
2
Create Bootable Media
Use tools like Rufus (Windows), Etcher (all platforms), or dd command to create bootable USB.
# Linux/macOS:
dd if=aegis-os-4.2.1-x86_64.iso of=/dev/sdX bs=4M
3
Boot from USB
Insert USB, restart computer, press F12/DEL/ESC (varies by system) to enter boot menu, select USB device.
4
Live Environment
Boot into Aegis OS live environment. Test hardware, wifi, and features before installing.
5
Run Installer
Click "Install Aegis OS" button on desktop. Installer will detect hardware automatically.
6
Partition & Format
Choose installation disk and partition scheme (automatic recommended). Optional: Enable full-disk encryption.
7
Install Base System
Installation proceeds automatically (5-15 minutes). Installer downloads and configures system packages.
8
Configure & Reboot
Set timezone, hostname, user account. Remove USB and reboot into installed Aegis OS system.

‚ùì Troubleshooting

Won't Boot: Check BIOS settings, disable Secure Boot temporarily, or try MBR boot mode.

Installation Fails: Verify ISO checksum, try different USB port, or recreate bootable media.

Hardware Issues: Boot into live environment to test and update drivers before full installation.

License Won't Activate: Verify internet connection and license code. Contact support if issues persist.
