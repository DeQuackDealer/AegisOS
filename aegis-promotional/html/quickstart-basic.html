Basic Edition - Quick Start (5 Minutes)

Step 1: Install (2 min)

sudo aegis-install basic && aegis-license activate YOUR_KEY

Step 2: Domain Setup (1 min)

aegis domain add example.com && aegis dns update --auto-ssl

Step 3: Office & Email (1 min)

Launch LibreOffice: libreoffice --writer

Configure Thunderbird: Click icon in taskbar (auto-configured)

Step 4: Security (1 min)

aegis security enable && aegis firewall auto-configure

✅ Done! Start Hosting Websites

Your domain is live, email is configured, office suite ready!
