Premium Analytics Endpoints

GET /api/v1/premium/analytics/advanced

Get advanced analytics with custom time ranges and dimensions.

curl -X GET http://localhost:5000/api/v1/premium/analytics/advanced \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "metric": "revenue", "start_date": "2025-01-01", "end_date": "2025-12-31", "dimensions": ["tier", "region", "source"], "format": "json" }'
GET /api/v1/premium/analytics/cohorts

Cohort analysis for user retention and lifecycle metrics.

curl -X GET http://localhost:5000/api/v1/premium/analytics/cohorts \ -H "X-API-Key: your-api-key"

Premium Reporting Endpoints

POST /api/v1/premium/reports/generate

Generate advanced reports with custom formatting.

curl -X POST http://localhost:5000/api/v1/premium/reports/generate \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "report_type": "comprehensive", "format": "pdf", "include_graphs": true, "export_data": true }'
POST /api/v1/premium/reports/schedule

Schedule automated report generation and delivery.

curl -X POST http://localhost:5000/api/v1/premium/reports/schedule \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "report_type": "executive_summary", "frequency": "monthly", "recipients": ["admin@example.com"], "format": "pdf" }'

Premium Webhook Endpoints

POST /api/v1/premium/webhooks/batch

Register multiple webhooks in a single request.

curl -X POST http://localhost:5000/api/v1/premium/webhooks/batch \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "webhooks": [ {"url": "https://example.com/webhook1", "events": ["payment.completed"]}, {"url": "https://example.com/webhook2", "events": ["license.renewed"]} ] }'

Premium License Management

POST /api/v1/premium/licenses/batch-issue

Issue multiple licenses in one request (Enterprise only).

curl -X POST http://localhost:5000/api/v1/premium/licenses/batch-issue \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "licenses": [ {"tier": "basic", "count": 10}, {"tier": "gamer", "count": 5} ], "duration_days": 365 }'
PUT /api/v1/premium/licenses/bulk-update

Update multiple licenses with new tiers or expiration dates.

curl -X PUT http://localhost:5000/api/v1/premium/licenses/bulk-update \ -H "X-API-Key: your-api-key" \ -d '{ "license_ids": ["lic-1", "lic-2", "lic-3"], "new_tier": "gamer" }'

Premium Monitoring & Alerts

POST /api/v1/premium/monitoring/create-alert

Create custom monitoring alerts with advanced conditions.

curl -X POST http://localhost:5000/api/v1/premium/monitoring/create-alert \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "name": "High CPU Usage", "metric": "cpu_usage", "condition": "greater_than", "threshold": 80, "duration_seconds": 300, "webhook_url": "https://example.com/alert" }'

Premium Data Export

POST /api/v1/premium/export/bulk

Export large datasets in multiple formats.

curl -X POST http://localhost:5000/api/v1/premium/export/bulk \ -H "X-API-Key: your-api-key" \ -d '{ "data_type": "users", "format": "csv", "include_metadata": true, "compression": "gzip" }'

Premium Team Management

POST /api/v1/premium/teams/create

Create and manage team accounts (Paid tiers only).

curl -X POST http://localhost:5000/api/v1/premium/teams/create \ -H "X-API-Key: your-api-key" \ -H "Content-Type: application/json" \ -d '{ "team_name": "Acme Corp", "billing_email": "billing@acme.com", "plan": "team_pro" }'

Premium Integrations

POST /api/v1/premium/integrations/configure

Configure third-party integrations (Slack, Discord, GitHub, etc).

curl -X POST http://localhost:5000/api/v1/premium/integrations/configure \ -H "X-API-Key: your-api-key" \ -d '{ "integration": "slack", "webhook_url": "https://hooks.slack.com/services/...", "events": ["payment.completed", "security.alert"] }'
