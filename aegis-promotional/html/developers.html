Build with Aegis OS APIs

Getting Started

Aegis OS provides 55+ REST API endpoints for licensing, tier management, security, automation, and more.

// Base URL https://aegis-os.com/api/v1 // Authentication X-Admin-Key: your-admin-key Authorization: Bearer token

Core Endpoints

GET /health
System health status and metrics
GET /api/v1/status
Detailed server status with uptime
GET /api/v1/tiers
All available tiers with pricing and features
GET /api/v1/tier/:name
Specific tier information

License Endpoints

POST /api/v1/admin/license/create
Create lifetime or recurring license
GET /api/v1/admin/licenses
List all licenses (admin only)
POST /api/v1/license/verify
Verify license validity
POST /api/v1/license/activate
Activate a license
DELETE /api/v1/admin/license/:id
Revoke a license (admin only)

Example Request

curl -X POST https://aegis-os.com/api/v1/admin/license/create \ -H "Content-Type: application/json" \ -H "X-Admin-Key: admin-secret-key-123" \ -d '{ "type": "lifetime", "tier": "basic" }'

Response Format

{ "license": { "id": "abc123def456", "tier": "basic", "type": "lifetime", "created": "2025-11-21T10:05:53", "status": "active", "activated": false } }
