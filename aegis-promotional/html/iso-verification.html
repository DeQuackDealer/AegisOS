üñ•Ô∏è Verification Instructions

Step 1: Windows (PowerShell)

Open PowerShell and run:

certUtil -hashfile C:\path\to\aegis-os-4.2.1-x86_64.iso SHA256

Compare the output with the checksum above.

Step 2: Linux/macOS (Terminal)

Run the following command:

sha256sum aegis-os-4.2.1-x86_64.iso
# or on macOS:
shasum -a 256 aegis-os-4.2.1-x86_64.iso

Verify the hash matches the official checksum.

üîë GPG Signature Verification

For Advanced Users:

gpg --verify aegis-os-4.2.1-x86_64.iso.asc

Expected output: "Good signature from Aegis OS Release"
