Frequently Asked Questions

Find answers to common questions

What is Aegis OS?
Aegis OS is a professional Linux distribution available in 5 editions: Freemium (free), Basic, Gamer, AI Developer, and Server. Each edition is optimized for specific use cases with tier-appropriate features.
Which edition should I choose?
Choose based on your needs: Freemium for basic Linux, Basic for security, Gamer for gaming, AI Developer for ML/AI, or Server for enterprise deployments. Use our comparison tool to find the best fit.
Can I upgrade tiers anytime?
Yes! Upgrade or downgrade at any time. You only pay the difference for the remaining period in your subscription cycle.
Is there a free trial?
Yes! All paid tiers include a 30-day free trial with full features. No credit card required. Plus, we offer a 30-day money-back guarantee.
How do I install Aegis OS?
Download the ISO image for your tier, create a bootable USB with Balena Etcher, and boot from it. The installer walks you through the setup process (typically 5-10 minutes).
Can I dual-boot with Windows/macOS?
Yes! Aegis OS can be installed on a separate partition alongside Windows or macOS. During installation, you can select which disk/partition to use and configure boot order.
Does Freemium have security features?
No. Freemium explicitly does NOT include security features. Security features (AI scanning, firewall, 2FA, audit logging) are exclusive to paid tiers (Basic and above).
What security features are in Basic?
Basic includes 7 security features: real-time threat scanning, AI-powered anomaly detection, UFW firewall with 23 rules, file integrity checking, network monitoring, 2FA support, and 30-day audit logging.
Can Basic run games?
Basic can run games, but for gaming optimization use the Gamer edition. Basic has standard graphics support, while Gamer has <5ms latency, 60+ gaming tools, and Vulkan 1.3/OpenGL 4.6.
What gaming tools are included?
Gamer includes Wine 8.21, Proton (latest), 1000+ game profiles, auto-game mode, Steam optimization, DXVK, VKD3D, and performance monitoring tools.
Does AI Dev include GPU support?
Yes! AI Dev includes CUDA 12.0 and cuDNN 8.6 pre-configured for GPU acceleration. It supports PyTorch, TensorFlow, and other frameworks with multi-GPU training.
What ML libraries are pre-installed?
AI Dev includes 50+ libraries: PyTorch 2.1, TensorFlow 2.14, JAX, scikit-learn, XGBoost, LightGBM, OpenCV, Hugging Face Transformers, NLTK, Librosa, and more.
Is Server edition headless?
Yes, Server is headless (no desktop). It boots in 25 seconds with only 512MB idle memory, perfect for servers. CLI and SSH are primary interfaces.
What's the Server edition uptime SLA?
Server guarantees 99.95% uptime (max 43 minutes downtime/year). Includes automatic failover, <5min recovery time, hourly backups, and 24/7 enterprise support.
Can Server handle 10,000+ concurrent users?
Yes! Server includes Nginx 1.25 (50,000+ req/sec), PostgreSQL 15 (10,000+ TPS), and supports 100,000+ concurrent connections with proper infrastructure.
Is there rebootless patching?
Yes! Server includes live kernel patching (kpatch). Security patches are applied automatically within 48 hours with zero downtime.
What backups are included?
Basic/Gamer: Daily backups (30-day retention). AI Dev: Daily backups (30-day retention). Server: Hourly backups (1-year retention) with PITR, cross-region replication.
Can I restore from backups?
Yes! You can restore full system, individual files, or use point-in-time recovery (Server only). Use the aegis-cli backup restore command.
What API rate limits apply?
Freemium: 100 req/hour. Basic/Gamer: 1,000 req/hour. AI Dev: 10,000 req/hour. Server: Unlimited. Plus 30 standard API endpoints.
