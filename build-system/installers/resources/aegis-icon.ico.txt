AEGIS ICON PLACEHOLDER
======================

This file is a placeholder for the aegis-icon.ico file.

To create the icon:
1. Create a 256x256 PNG image with your Aegis OS logo
2. Convert to ICO format with multiple sizes:
   - 16x16
   - 32x32
   - 48x48
   - 64x64
   - 128x128
   - 256x256

Recommended tools:
- GIMP (free): Export as .ico
- ImageMagick: convert logo.png -define icon:auto-resize=256,128,64,48,32,16 aegis-icon.ico
- Online: https://convertio.co/png-ico/

Requirements:
- Format: Windows ICO
- Color depth: 32-bit with alpha transparency
- Minimum size: 256x256 for high DPI displays

Place the finished aegis-icon.ico file in this directory (resources/).
Delete this .txt placeholder file after adding the real icon.
