# Aegis OS - Build System

## Quick Start

```bash
chmod +x build-aegis.sh
./build-aegis.sh
```

Choose an edition (1-6) and the build runs automatically. Takes 1-2 hours.

## Requirements

- Linux VM (Ubuntu 20.04+)
- 50GB+ disk space
- 8GB+ RAM
- Internet connection

## What You Get

- **Freemium** (FREE) - Base Linux + gaming
- **Basic** ($49) - Security features
- **Gamer** ($99) - Gaming optimized
- **AI Dev** ($149) - ML/AI pre-installed
- **Server** ($199) - Enterprise
- **Workplace** ($79) - Teams/Office365

After build, use Balena Etcher to flash the ISO to USB.

## Folders

- `aegis-os-*` - Complete build system for each tier
- `aegis-promotional` - Website (optional)
- `build-aegis.sh` - Main build launcher
