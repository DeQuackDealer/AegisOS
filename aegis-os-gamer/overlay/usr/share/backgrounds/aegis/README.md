# Aegis OS Wallpapers

Place wallpaper images here. Required files:

- `aegis-win11-dark.png` - Dark Windows 11-style wallpaper (1920x1080 or higher)
- `default.jpg` - Default wallpaper fallback

The Aegis-Win11 theme uses dark colors (#202020 background) so dark wallpapers work best.

## Recommended Style
- Dark gradient backgrounds
- Subtle geometric patterns
- Aegis branding elements (optional)
- 4K resolution (3840x2160) for best quality
